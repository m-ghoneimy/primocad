Z-Wizard

Z-Wizard is a Autodesk® Civil 3D® add-in application that facilitates the creation of objects with elevation values. It contains two panels of tools: the Text Tools panel, and the Surface Tools panel. The Text Tools panel contains commands related to Autodesk® AutoCAD® text objects which have numeric Z values as contents. The Surface Tools panel contains commands related to the Civil 3D surfaces.

  • Productive commands
  • Performance optimized
  • Compatible with Civil 3D 2021-2026
