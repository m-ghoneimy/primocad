Duration Calculator Documentation

Overview

The Duration Calculator is a powerful tool designed for construction and project management professionals to calculate the time required to complete activities based on production rates, resources, and efficiency factors.

Formula

The calculator uses the following fundamental formula:

Duration = Production Quantity / (Producer Rate × Number of Producers × Efficiency)

Formula Components

  • Production Quantity: Total amount of work to be completed
  • Producer Rate: Output per producer per time unit
  • Number of Producers: Total resources working on the activity
  • Efficiency: Performance factor (0.01 to 1.00, where 1.00 = 100%)

Getting Started

Basic Workflow

  1. Configure Global Settings - Set your units and decimal precision
  2. Enter Activity Data - Input production details
  3. Calculate Duration - Get your results instantly
  4. Review Summary - Check detailed breakdown
  5. Edit if Needed - Make adjustments and recalculate

Global Settings

Quantity Unit

Select the unit for measuring production quantity:

  • Cubic Meter (m³): For volume-based activities (concrete, excavation)
  • Square Meter (m²): For area-based activities (flooring, painting)
  • Meter (m): For linear activities (piping, cabling)
  • Ton: For weight-based activities (steel, materials)
  • Kilogram: For smaller weight measurements
  • Custom: Define your own unit (e.g., pieces, panels, doors)

Example: If you're calculating concrete pouring time, select "Cubic Meter (m³)"

Time Unit

Select the unit for measuring time duration:

  • Hour: For short-duration activities
  • Day: Most common for construction activities
  • Week: For longer planning horizons
  • Month: For large-scale projects
  • Custom: Define your own unit (e.g., shift, cycle)

Example: For daily planning, select "Day"

Decimals

Choose the decimal precision for all numerical results (0-3 decimal places):

  • 0 decimals: Whole numbers (e.g., 10 days)
  • 1 decimal: One decimal place (e.g., 10.5 days)
  • 2 decimals: Two decimal places (e.g., 10.25 days)
  • 3 decimals: Three decimal places (e.g., 10.125 days)

Recommendation: Use 2 decimals for most construction activities

Custom Units

When you select "Custom" for either Quantity Unit or Time Unit:

  1. A text input field appears below the dropdown
  2. Enter your custom unit name (e.g., "pieces", "shifts", "cycles")
  3. Labels throughout the calculator automatically update to reflect your custom unit
  4. All calculations and displays will use your custom unit

Activity Data

Name

Enter a descriptive name or code for the activity.

Examples:

  • "Foundation Excavation"
  • "Concrete Pour - Level 1"
  • "ACT-001"
  • "Brick Wall - North Façade"

Tip: Use clear, consistent naming conventions for easy tracking

Production Quantity

Enter the total amount of work to be completed, measured in your selected quantity unit.

Examples:

  • 150.00 m³ (cubic meters of concrete)
  • 500.00 m² (square meters of flooring)
  • 2500.00 m (meters of piping)
  • 45.00 ton (tons of steel)

Important: Must be a positive number greater than 0

Producer Rate

Enter the production rate per producer per time unit. This represents how much work one producer can complete in one time unit.

The label dynamically updates based on your selected units. For example:

  • "Producer Rate (m³/day per producer)" if using cubic meters and days
  • "Producer Rate (m²/hour per producer)" if using square meters and hours

Examples:

  • 5.00 m³/day (one crew can pour 5 cubic meters per day)
  • 25.00 m²/day (one painter can cover 25 square meters per day)
  • 100.00 m/day (one team can install 100 meters of pipe per day)

Tip: Base this on historical data or industry standards for accuracy

Number of Producers

Enter the total number of producers (crews, workers, machines) working on this activity.

Examples:

  • 2 (two crews working simultaneously)
  • 3.5 (three full crews and one half crew)
  • 1 (single resource)

Note: Can be decimal to account for partial resources (e.g., 1.5 crews)

Efficiency

Enter the efficiency factor as a decimal between 0.01 and 1.00, where:

  • 1.00 = 100% efficiency (ideal conditions)
  • 0.90 = 90% efficiency (typical for good conditions)
  • 0.75 = 75% efficiency (accounting for delays)
  • 0.50 = 50% efficiency (difficult conditions)

Common Efficiency Factors:

  • 0.95-1.00: Repetitive work, experienced crew, ideal conditions
  • 0.85-0.90: Normal construction conditions
  • 0.75-0.80: Some obstacles or learning curve
  • 0.60-0.70: Complex work, limited access, weather impacts
  • Below 0.60: Difficult conditions, significant constraints

Example: If you expect 85% efficiency due to normal site conditions, enter 0.85

Using the Calculator

Step-by-Step Example

Let's calculate the duration for a concrete pouring activity:

Scenario: Pour 150 cubic meters of concrete with 2 crews

Step 1: Global Settings

  • Quantity Unit: Cubic Meter (m³)
  • Time Unit: Day
  • Decimals: 2

Step 2: Activity Data

  • Name: Foundation Slab Pour
  • Production Quantity: 150.00 m³
  • Producer Rate: 10.00 m³/day per producer
  • Number of Producers: 2
  • Efficiency: 0.85 (85% efficiency)

Step 3: Calculate

  • Click "Calculate" button
  • Duration = 150 / (10 × 2 × 0.85)
  • Duration = 150 / 17 = 8.82 days

Step 4: Review Summary

The summary shows:

  • Activity Name: Foundation Slab Pour
  • Production Quantity: 150.00 m³
  • Producer Rate: 10.00 m³/day
  • Number of Producers: 2
  • Efficiency: 85.00%

Tips & Best Practices

Accuracy Tips

  1. Use Historical Data: Base producer rates on actual past performance when possible
  2. Account for Learning Curves: Use lower efficiency for new or complex activities
  3. Consider Site Conditions: Adjust efficiency for access, weather, and other constraints
  4. Include Setup Time: Factor in mobilization and setup in your rates
  5. Regular Updates: Update rates based on actual performance for future accuracy

Frequently Asked Questions

Q: Can I use partial producers (e.g., 2.5 crews)?

A: Yes, the Number of Producers field accepts decimal values to account for partial resources.

Q: What efficiency factor should I use?

A: Start with 0.85 for typical conditions. Adjust based on your experience, site conditions, and activity complexity.

Q: Can I save my calculations?

A: The current version doesn't save automatically. We recommend taking screenshots or noting down the summary for your records.

Q: How accurate are the results?

A: Accuracy depends on the quality of your input data. Using historical performance data and realistic efficiency factors produces the most accurate results.

Q: Can I use this for non-construction activities?

A: Yes! The calculator works for any activity where you have production rates and resources. Use custom units to adapt it to your needs.

Q: What's the difference between producer rate and production quantity?

A: Production quantity is the total work to be done. Producer rate is how much one producer can complete per time unit.

Q: How do I account for multiple shifts?

A: Adjust your number of producers to reflect total working shifts (e.g., 2 crews × 2 shifts = 4 producers) or use custom time units like "shift".

Formula Derivation

Understanding the formula helps ensure correct usage:

Duration = Production Quantity / (Producer Rate × Number of Producers × Efficiency)

Why this formula?

  1. Producer Rate × Number of Producers = Total production capacity per time unit
  2. × Efficiency = Adjusted for real-world conditions
  3. Production Quantity ÷ Adjusted Capacity = Time required

Example Calculation:

  • Production Quantity: 100 m³
  • Producer Rate: 10 m³/day per producer
  • Number of Producers: 2
  • Efficiency: 0.80 (80%)
Total Capacity = 10 × 2 = 20 m³/day
    Adjusted Capacity = 20 × 0.80 = 16 m³/day
    Duration = 100 ÷ 16 = 6.25 days
